About

This application searches for links in an html file and lists them as internal and external links.

Usage

To search for links in a source provide the url including protocol to that source, for example: http://example.com. Information such as filename (including path), queries and more are optional. Please Notice:

  • Lower level subdomains will be categorized as internal links.
  • Same level subdomains will be categorized as external links.
  • Anchor links to the same page will not be recognized as links.
  • External and internal link differentiation is based on inclusion of https http or www in the link. Therefore it is possible that some links may we wrongly classified as internal link.
  • The links in the internal links result set function correctly only if the protocol is http or https.
